# Example Prompts for Red Hat Knowledge Base MCP Server

This document provides example prompts that demonstrate how AI assistants can use the Red Hat Knowledge Base MCP Server tools effectively.

## Knowledge Base Search

### Search for RHEL upgrade issues
> "Search the Red Hat Knowledge Base for known issues when upgrading from RHEL 8 to RHEL 9. What are the most common problems?"

### Find solution for kernel panic
> "I'm seeing kernel panics on my RHEL system. Search the Red Hat Knowledge Base for solutions related to kernel panic crashes."

### Troubleshoot OpenShift errors
> "Search the Red Hat Knowledge Base for solutions to `ImagePullBackOff` errors in OpenShift. What are the common causes and fixes?"

## RHEL System Administration

### Package management issues
> "Search for known issues with DNF package management in RHEL 9. Are there any common dependency resolution problems?"

### Networking troubleshooting
> "Find Red Hat KB articles about NetworkManager connectivity issues in RHEL. How do I debug DNS resolution failures?"

### SELinux problems
> "Search for solutions to SELinux denials affecting my web server. How do I properly configure SELinux policies?"

## OpenShift and Kubernetes

### Cluster installation issues
> "Search for known issues during OpenShift cluster installation. What are the common failure points?"

### Operator troubleshooting
> "Find KB articles about troubleshooting degraded cluster operators in OpenShift."

### Storage issues
> "Search the Red Hat Knowledge Base for persistent volume provisioning failures in OpenShift."

## Security and Compliance

### CVE information
> "Search for Red Hat KB articles about recent CVEs affecting RHEL 9. What patches are available?"

### Certificate management
> "Find solutions for certificate expiry issues in OpenShift. How do I rotate certificates?"

## Performance

### System performance tuning
> "Search for Red Hat KB articles about performance tuning for RHEL systems running database workloads."

### Container performance
> "Find KB articles about optimizing container performance in OpenShift. What resource limits are recommended?"

## Tips for Effective Prompts

- **Be specific about the product** - Mention RHEL version, OpenShift version, or specific product names for more targeted results.
- **Include error messages** - When troubleshooting, include the exact error message or code for better search results.
- **Describe the context** - Mention what you were doing when the issue occurred to help narrow down relevant articles.
- **Ask for workarounds** - Red Hat KB articles often include both fixes and temporary workarounds.
